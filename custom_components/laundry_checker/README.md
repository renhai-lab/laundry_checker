## Laundry Checker Configuration Options

### Available Options
- **Location**: The location for weather data.
- **API Host**: The host for the weather API (default: `DEFAULT_QWEATHER_API_HOST`).
- **Max AQI**: Maximum Air Quality Index (default: `DEFAULT_MAX_AQI`).
- **Min Suitable Hours**: Minimum hours suitable for laundry (default: `6`).
- **Max Suitable Humidity**: Maximum humidity suitable for laundry (default: `85.0`).
- **Start Hour**: Start hour for laundry (default: `6`).
- **End Hour**: End hour for laundry (default: `22`).
- **Preferred End Hour**: Preferred end hour for laundry (default: `18`).
- **Unsuitable Weather Types**: List of unsuitable weather types (default: empty list).
